# AngularMovieApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.10. 
I created a two version of movie application. Using Rxjs and Signals.

## Angular Movie App
* Angular
* Angular Material
* PrimeNG
* RxJS
* Ngx Skeleton Loader
* Tailwind CSS
* Json Server

## Build

Run `npm run dev` to build the project.
